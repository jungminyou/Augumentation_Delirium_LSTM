# Manuscript-to-Code Mapping (Journal Submission)

## Purpose

This document maps each manuscript section to executable modules in this repository.

## Data Privacy Statement

- The real clinical dataset used for manuscript analysis is private and contains sensitive personal information.
- Raw patient-level source data is not distributed in this repository.
- Public release must include only de-identified, approved, and policy-compliant artifacts.

## Section Mapping

### 1. Study Data and Leakage-Free Cohort Design

- Manuscript content:
  - Longitudinal cohort construction.
  - Leakage control and exclusion of leakage-prone variables.
  - Patient-level split (train/validation/test).
- Code mapping:
  - src/augmentation.py
  - run_stage_01_augmentation.py
- Key implementation points:
  - Drops leakage feature `CAM_diagnosis` if present.
  - Splits by `pid` instead of row-wise random split.

### 2. Minority Trajectory Augmentation (TimeGAN + DTW-SMOTE)

- Manuscript content:
  - Synthetic sequence generation for minority class.
  - TimeGAN pretraining and sample generation.
  - DTW-aware interpolation.
- Code mapping:
  - src/augmentation.py
  - run_stage_01_augmentation.py
- Key implementation points:
  - `TimeGAN` class for sequence embedding/recovery/generation.
  - `dtw_smote_pair` and `dtw_smote` for alignment-aware interpolation.

### 3. Distribution-Constrained Reconstruction

- Manuscript content:
  - Range and type constraints on synthetic features.
  - Label pattern sampling from real minority patients.
- Code mapping:
  - src/augmentation.py
- Key implementation points:
  - `enforce_distribution_strict` for min/max clipping and integer snapping.
  - `rebuild_dataframe_refined` for realistic label sequence reconstruction.

### 4. Tensor Conversion and Leakage-Safe Scaling

- Manuscript content:
  - Conversion of tabular trajectories to 3D tensors.
  - Scaling fit only on original train set.
- Code mapping:
  - src/tensor_prep.py
  - run_stage_02_tensor_prep.py
- Key implementation points:
  - `df_to_3d_tensor`.
  - `build_scaled_tensors` with train-only fitting and shared transform.

### 5. Multi-Task Soft-Gated Modeling

- Manuscript content:
  - Soft-gated architecture linking occurrence and subtype/severity predictions.
  - Baseline and augmented training settings.
- Code mapping:
  - src/models.py
  - src/train.py
  - run_stage_03_train_eval.py
- Key implementation points:
  - `build_multi_output_lstm_gated`.
  - Logistic regression baseline with gating.
  - LSTM models for original, DTW-SMOTE, and TimeGAN datasets.

### 6. Performance Evaluation

- Manuscript content:
  - AUROC/AUPRC with confidence intervals.
  - Calibration and decision curve analysis.
  - Outcome-specific metric tables.
- Code mapping:
  - src/evaluate.py
  - run_stage_03_train_eval.py
- Key implementation points:
  - `compute_auc_prc_with_ci` (bootstrap).
  - `calc_calibration_metrics`.
  - `calculate_net_benefit`.

### 7. Fidelity and Statistical Validation

- Manuscript content:
  - Pairwise statistical comparison (genuine delirium in original train vs synthetic cohorts).
  - JSD, DTW, and Wasserstein distance.
  - t-SNE fidelity visualization.
- Code mapping:
  - src/fidelity.py
  - run_stage_04_fidelity.py
- Key implementation points:
  - `run_pairwise_statistics`.
  - `compute_jsd_table`.
  - `compute_distance_metrics`.

- Random seed: 42
- Split policy: patient-level (`pid`)
- Leakage control: explicit feature drop and train-only scaling fit
- Entry points:
  - run_all.py (full pipeline)
  - run_stage_01_augmentation.py to run_stage_04_fidelity.py (section-wise)
